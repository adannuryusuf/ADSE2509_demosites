CSS Animation

CSS animation is the process of changing an elementproperty over a given duration of time without the need for jvascript. You can change an element size color location on screen and even rotate it to name a few. To create a css animation you need to define properties to be animated anf the keyframaes which define how the animated element will display at a given time.
